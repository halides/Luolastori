# Luolastori
tiralabra K2021

Developed on Ubuntu 20.04 Focal Fossa

```
sudo apt install g++ libcppunit-dev
make && ./luolastori
make test && ./test_luolastori
make clean
```

to create coverage report:
```
./create_coverage.sh
```

Edit headers/luolastori.hpp defines to change size of room, rerun make etc.
